# 🧠 AI Trip Itinerary Planner

An AI-powered travel planner that generates a personalized itinerary based on your destination and travel dates. Built using **LangChain** for AI agent logic, **Flask** for the backend API, and **React** for the frontend.

---

## 🚀 Features

- 🌍 Intelligent trip itinerary generation
- 📅 Input: Place, Start Date, and End Date
- 🤖 LangChain-powered reasoning agent
- 🔗 Flask-based backend API
- 💻 Modern React UI
- 📄 Generates downloadable PDF of the itinerary
